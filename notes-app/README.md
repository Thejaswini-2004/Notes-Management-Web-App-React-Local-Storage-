
A simple and responsive Notes Management Web Application built using React.js.  
This app allows users to create, edit, delete, and store notes using browser localStorage.

 Features

-  Create new notes
-  Edit existing notes
-  Delete notes
-  Persistent storage using localStorage
-  Sidebar for notes list
-  Separate editor view for writing notes

Tech Stack

- Frontend: React.js
- State Management: React Hooks (useState, useEffect)
- Storage: Browser LocalStorage
